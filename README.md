### Ziqi Liu

Ph.D student at Wuhan University. Working on code generation for geoscience tasks using large language models — mostly thinking about how to make spatial analysis less painful for people who aren't programmers.

Research tends to involve a lot of prompt engineering, staring at model outputs, and slowly accepting that evaluation is the hard part.

&nbsp;

**Interests** &nbsp; LLM · Code Generation · GIS · Remote Sensing · Spatial Intelligence
